Apa itu HTML???

(Hypertext Markup Language) adalah bahasa markup standar untuk membuat dan menyusun halaman dan aplikasi web. HTML dibuat oleh Tim Berners-Lee, seorang ahli fisika di lembaga penelitian CERN yang berlokasi di Swiss. Versi pertamanya dirilis pada tahun 1991, dengan 18 tag.
Bagaimana Cara Kerja HTML? File HTML diakhiri dengan ekstensi .html atau .htm. Ekstensi file ini bisa dilihat dengan mengunakan web browser apa pun (seperti Google Chrome, Safari, atau Mozila Firefox).
